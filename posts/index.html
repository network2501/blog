Junos EX Switch Upgrade via USB

I have a Juniper EX2200c switch that needs upgrading. So here goes the quick and dirty run down. Jump into the shell. root> start shell Make a folder to mount your FAT16/32 USB drive, then mount it. root> start shell root@:RE:0% mkdir /var/tmp/usb root@:RE:0% mount -t msdosfs /dev/da1s1 /var/tmp/usb Jump back to the CLI and install your new Junos. root> start shell root@:RE:0% cli {master:0} root> request system software add /var/tmp/usb/jinstall-ex-2200-12.
